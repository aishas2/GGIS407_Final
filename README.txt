This repository contains the Jupyter Notebook code and files for the investigation of the change in crop class cover in California from 2014 to 2016 in accordance with the standards of University of Illinois Urbana-Champaign's GGIS 407: CyberGIS & Geospatial Data Science in Spring 2022 taught by Dr. Shaowen Wang. 
Data sources: California Open Data Portal, Land IQ, LLC, US Census Bureau
